🚀 Hướng dẫn deploy lên GitHub Pages

  1. Tải file eGreen_v3.html về máy
  2. Vào github.com → tạo repo mới (ví dụ: egreen-bdsc)
  3. Upload file HTML → đặt tên là index.html
  4. Vào Settings → Pages → Source: main / root
  5. GitHub tự tạo link: https://[user].github.io/egreen-bdsc
  6. Chia sẻ link cho cả đội dùng
